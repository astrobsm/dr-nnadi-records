DEPLOY WITHOUT GIT - SIMPLE METHOD
========================================

Since Git is not installed on your computer, follow these steps to deploy using GitHub's web interface:

STEP 1: Go to Your Repository
==============================
1. Open your browser
2. Go to: https://github.com/astrobsm/dr-nnadi-records
3. Sign in to GitHub if you haven't


STEP 2: Upload Files
==============================
1. Click the "Add file" button (top right)
2. Select "Upload files"
3. Drag and drop these 6 files from your folder:
   
   ‚úì index.html
   ‚úì styles.css
   ‚úì app.js
   ‚úì manifest.json
   ‚úì logo.png
   ‚úì README.md

4. In the commit message box, type: "Deploy application"
5. Click "Commit changes" (green button)


STEP 3: Enable GitHub Pages
==============================
1. Click "Settings" tab (top of page)
2. Scroll down and click "Pages" in the left sidebar
3. Under "Source":
   - Select branch: "main"
   - Select folder: "/ (root)"
4. Click "Save"
5. Wait 1-2 minutes


STEP 4: Access Your Application
==============================
Your app will be live at:

   üåê https://astrobsm.github.io/dr-nnadi-records/

Copy this URL and:
- Open it in your phone's browser
- Add to Home Screen to use like an app!

========================================

To update the app with changes:

1. Go to: https://github.com/astrobsm/dr-nnadi-records
2. Click on the file you want to update (e.g., app.js)
3. Click the pencil icon (‚úèÔ∏è Edit)
4. Make your changes
5. Scroll down and click "Commit changes"
6. Changes appear in 1-2 minutes at your live site


========================================
ALTERNATIVE: INSTALL GIT (OPTIONAL)
========================================

If you want to use Git in the future:

1. Download Git: https://git-scm.com/download/win
2. Install with default settings
3. Restart your computer
4. Run auto-deploy.bat again
